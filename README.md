# Black MIDI Software Wiki (or Blog)

These collection of pages are intended to gather knowledge base of software/tools related to production or visualization of Black MIDIS.

## Reading the pages

To read the pages, knowledge and comprehension of the English language is required.

## Running the site locally

1. Install node.js
2. Add the node executable to PATH (e.g. add `node.exe` to Environment Variables' Path property)
3. Clone/download this repository.
4. Set the repository as current directory in a terminal (e.g. through Command Prompt):
```
cd %userprofile%/Downloads/b
```
5. Install dependencies:
```
npm install
```
6. Run the server:

```
npm run docs:dev
```

7. Visit the displayed link using your browser.
